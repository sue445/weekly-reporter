play環境構築---------最新版を落とすhttp://www.playframework.org/download---------http://d.hatena.ne.jp/daicham/20110921/1316600869を参考に~/Library/Javaに入れる---------githubのリポジトリhttps://github.com/sue445/weekly-reporter---------http://change-the-world.heroku.com/heroku周りのインストールをする---------ローカルにcloneするgit clone git@github.com:sue445/weekly-reporter.git---------Macのgithubクライアントでリポジトリ管理するとherokuにデプロイできなくなるっぽいのでやらないほうがいい（git push heroku masterしてもローカルのgitリポジトリにcommitされるだけ・・・）---------play ecでeclipse周りの設定を行う（http://www.playframework.org/documentation/1.0.1/ideに、eclipse関係の設定はチームで共有するなとあるのでgitignoreしてます）----------gitのremoteにherokuを追加git remote add heroku git@heroku.com:weekly-reporter.git----------gitにデプロイするgit remote heroku master
